# Gulp Snippets

- [gulp.src()]()
- [gulp.task()]()
- [gulp.watch]()

## gulp.src()

**Comand:** `src`

```
gulp.src('route')
```

## gulp.task()

**Comand:** `task`

```
gulp.task('name', ['tasks'], function);
```

## gulp.watch()

**Comand:** `watch`

```
gulp.watch(path, ['tasks']).on('event', function);
```

## pipe()

**Comand:** `pipe`

```
.pipe(function)
```
